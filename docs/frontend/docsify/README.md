## docsify介绍

- Docsify即时生成您的文档网站。与 GitBook 不同，它不会生成静态 html 文件。相反，它会智能地加载和解析您的 Markdown 文件，并将它们显示为网站。



- docsify官方文档入口:[https://docsify.js.org/](https://docsify.js.org/)
